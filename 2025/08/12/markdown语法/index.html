Markdown 常见语法

标题

Markdown 使用 # 符号来表示标题，支持 六级标题， # 的数量代表标题的级别。

“ # ”——一级标题

” ## ”——二级标题

其他同理

加粗

“ ** ”括住要加粗的部分即可实现加粗

eg: 你好

高亮

”==“括住要进行强调的部分即可实现高亮

eg: 你好

斜体

“ * ”括住要进行斜体的部分即可实现文本斜体。

eg: 哈哈哈

中划线（删除线）

” ~~ ”括住要中划的部分即可。

eg: 会魔法的麻瓜

分割线

使用三个或更多的 - 、 * 或 _ 创建分隔线。

列表

  • 无序列表

    使用 - 、 * 或 + 作为标记符，通过“ * ”加空格实现，例如：

    • 富士康 i 哦物品

  • 有序列表

​ 使用数字加点 . ，数字顺序可以不连续，最终会按顺序渲染。 例如：

1.   哈哈哈
2.   嘤嘤嘤 
   1. 呜呜呜

要结束一段列表，只需连续按 两次 回车键 。

  • 打卡框

使用“ * 【 】 ”来实现，如果要打勾，只需将方括号的空格换成 x(大小写不限)。

  • [ ] 今日计划
  • [x] 练琴

代码块

  • 整段代码块

段首及段尾用” ``` “即可, 例如：

  • 行内代码块/注释

将选中的内容用“ ` ”框起来即可。

eg: 跳舞的梵谷

引用

用” > “放在被引用的内容的段首即可，例如：

人间的青草地

​ 需要浇水

​ 内心的花园就

​ 不会枯萎~

​ ——方大同《每个人都会》

注释：要结束一段引用，只需连续按 两次 回车键 。引用可进行嵌套。

窗外的麻雀 在电线杆上多嘴~

    	——jay chou《七里香》

超链接

将选中的文本用” [] “ 括起来，并在后面用” ()“ 标明跳转地址。

eg: 如果有问题，就找度娘。

  • 超链接标签复用

实现效果：百度其实就是度娘

  • 注脚

将选中的文本用“ [] ”框起来，在后面用“ [^] “标注注脚。

eg: 周[^ 1]杰伦[^ 2]

[^ 1]: jay chou
[^ 2]: 最喜博主爱的华语歌手

图片

添加图片与链接类似，只需在前面加一个 ! 。这里的图片可以是本地，也可以是网络（图床） 。

表格

使用 | 和 - 创建表格。 CTRL + enter 切换到下一数据行。

表格 表头
哈哈哈 嘻嘻嘻
嚯嚯嚯 嘤嘤嘤

数学公式

Typora 的数学公式有两种：行中公式和独立公式。行中公式放在文中与其它文字混编，独立公式单独成行。

  • 行内公式：用“ $ ”括起来
  • 独立公式：用‘’$$‘’括起来

如何输入上下标:

^ 表示上标, _ 表示下标。如果上下标的内容多于一个字符，需要用 {} 将这些内容括成一个整体。上下标可以嵌套，也可以同时使用。
